# Assignment Planner
## GUI Database Final

The purpose of this program is to help students stay organized by allowing them to add and remove assignments into a planner.

This system was created with:
* Derby Embedded Database
* Java
* JavaFX
* Scenebuilder
* Intellij IDEA
